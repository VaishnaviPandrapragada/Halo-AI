# HALO

HALO is an AI-powered safety companion designed to provide real-time guidance, emergency support, and practical safety assistance.

## Features

* AI-powered safety guidance using Gemini
* Emergency contact management
* SOS activation workflow
* Safety-focused conversational assistant
* Mobile-friendly interface

## Tech Stack

* React
* TanStack Start
* Vite
* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* Google Gemini API

## Running Locally

### Backend

```bash
cd backend
npm install
npm run dev
```

Create a `.env` file:

```env
GEMINI_API_KEY=your_api_key
MONGO_URI=your_mongodb_uri
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
