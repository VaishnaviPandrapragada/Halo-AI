# HALO

HALO is an AI-powered safety companion designed to provide real-time guidance, emergency support, and practical safety assistance.

## Features

* AI-powered safety guidance using Gemini
* Emergency contact management
* SOS activation workflow
* Safety-focused conversational assistant
* Mobile-friendly interface

## Tech Stack

* React
* TanStack Start
* Vite
* Node.js
* Express.js
* MongoDB Atlas
* Mongoose
* Google Gemini API

## Running Locally

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Setup requires:

- GEMINI_API_KEY
- MONGO_URI

Create a .env file with these variables before running the project.
